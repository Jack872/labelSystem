# springboot重构后端

#### 介绍
这是一个重构后端项目，我们将原有的Koa后端重构为SpringBoot后端。
#### 软件架构
本项目的目标是将现有的Koa后端重构为SpringBoot后端。这样做的目的是为了未来能够承载更多的应用。


#### 安装教程

1. 克隆项目到本地
2. 使用IDEA打开并导入项目
3. 安装所需的依赖
4. 运行项目
5. 配置数据库
